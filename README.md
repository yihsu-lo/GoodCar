## GoodCar iOS
>An e-commerce platform for second-handed vehicles.

### Feature
  * Home
    * Login Facebook / View As Guest
    * Upload Car Information and Picture
    * Find Car with Different Criteria
    * Share Car via Facebook
    * Like Car
    * Comment on the Car
  * Profile
    * My Selling Cars
    * My Loved Cars
  * Comment
    * Check Comments Which You Involved

### Libraries
  * FBSDK
  * FirebaseSDK
  * Haneke
  * Tracking Tool
    * Firebase Analytics
    * Crashlytics
